### Quick Start

> [!NOTE]
> This Application is still in development and might not be stable, yet.

**1. Clone repositories**

```sh
git clone https://github.com/adhxabre/re-astrvon
cd re-astrvon
```

**2. Install dependencies**

```sh
npm install
```

or

```sh
pnpm install
```

or

```sh
yarn install
```

**3. Run**

> [!TIPS]
> You can run it with `node src/index.ts` without having to compile it first!

Run the bot using:

```shell
npm run dev
```

or

```shell
pnpm dev
```

or

```shell
yarn dev
```

Also, you can run it on your server using PM2 by running this command:

```shell
pm2 src/index.ts
```

> [!INFO]
> A full commands can be found at our website. But, since it's still in development, you can visit the command files one
> by one (even tho we didn't suggest you to do it, but it's the only way)
